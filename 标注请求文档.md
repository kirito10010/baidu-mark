## 任务管理

URL：http://10.212.80.215:8901/api/marking/listtask
请求：POST
请求参数：

```json
{
    "page":1,//当前页码，从1开始
    "rows":2,//每页显示的记录数
    "active":1,//常见含义是“是否激活/有效”，通常为 1（有效）、0（无效/删除）或 null（全部）
    "id":null,//任务id，格式["113817","113817"]
    "name":null,//任务名称（模糊查询）
    "opid":null,//作业员id
    "qaid":null,//质检员id
    "flow":null,//流程状态/阶段
    "status":null//任务状态
}
```

{

## 类型管理

URL：http://10.212.80.215:8901/api/marking/listtype
请求：POST
请求参数：（全部父类请求）

```json
{
    "parent":0,//查询父类数据，所有的父类都是0
    "active":1//是否有效/启用-------------------暂时不理解
}
```

返回参数示例：

```json
{
    "errno": 0,
    "msg": "success",
    "data": [
        {
            "id": "0",
            "skey": " ",
            "name": "顶级分类",//父类名字
            "color": "#ffffff",//父类颜色
            "typeid": "000000",//任务大类id/任务类型编码
            "parent": "",//父任务ID（这是祖宗最高辈分）
            "active": "1"//是否有效/启用-------------------暂时不理解
        }
    ],
    "total": 1
}
```

URL：http://10.212.80.215:8901/api/marking/listtype
请求：POST
请求参数：（某个父类下的所有数据---代指：项目的框名称数据）

```json
{
    "sort":"typeid",//请求参数名，通常表示 排序字段
	"parent": "900000",//父类id
}
```

{
